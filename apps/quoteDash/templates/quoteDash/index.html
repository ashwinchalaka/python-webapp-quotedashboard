Welcome to the Quote Dashboard

This application was built using Python and an MTV framework called Django. The pages are of course organized and styled with HTML5 and CSS. There is also some styling with Bootstrap.

Built from scratch by Ashwin Chalaka.

Post Quotations

Using this application, you'll be able to add, view, like, and remove quotes.

Quotations By User

Users will be able to check other user's pages who have posted and be able to see all that user's quotations.

Edit User Information

Users will be able to edit their information (first name, last name, email, but not their password)
